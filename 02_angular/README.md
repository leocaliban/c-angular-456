# Angular

Criar aplicação angular a partir de uma versão CLI local

1. Criar uma pasta para conter todas as aplicações da versão local.
2. Nessa pasta criar um novo projeto NPM:
   * `npm init`
3. Instalar o Angular CLI da versão desejada localmente:
   * `npm install @angular/cli@1` *> versão 1*
4. Deletar o arquivo package.json.
5. Criar a nova aplicação na versão local do CLI:
   * `ng new app-angular --prefix=apg`
